## 脚本介绍
本脚本基于[CoiaPrant/MediaUnlock_Test](https://github.com/CoiaPrant/MediaUnlock_Test)代码进行修改

+ 优化了Disneyplus的判断准确性

+ 新增MyTVSuper解锁判断

+ 新增Dazn解锁判断

+ 新增Hulu Japan解锁判断

+ 新增赛马娘解锁判断

+ 新增舰娘解锁判断

## 使用方法

**使用脚本前请确认curl已安装**

````bash
bash <(curl -L -s https://raw.githubusercontent.com/lmc999/RegionRestrictionCheck/main/check.sh)
````
