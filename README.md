## 使用方法

**使用脚本前请确认curl已安装**

````bash
bash <(curl -L -s https://raw.githubusercontent.com/lmc999/RegionRestrictionCheck/main/check.sh)
````

**或者直接运行以下Docker命令**
````docker
docker run --rm --net=host lmc999/regioncheck
````
